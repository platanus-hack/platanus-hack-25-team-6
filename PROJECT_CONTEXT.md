# Senior Wellness Platform - Project Context

## Overview
AI-powered wellness companion for seniors focusing on mobility and at-home exercise routines.

## Core Features

### 1. Initial Assessment
- Evaluate user's current mobility level
- Generate personalized exercise plan based on capabilities
- Focus on home-friendly exercises

### 2. AI Exercise Monitoring
- Real-time pose detection via phone camera
- Performance tracking and analysis
- Form correction and feedback

### 3. AI Companion Features
- Video generation with exercise examples
- Voice feedback using AI (ElevenLabs)
- Exercise reminders and scheduling
- Progress tracking over time

### 4. User Experience
- PWA for easy access (no app store needed)
- Simple, senior-friendly interface
- Works on phone/tablet

## Target Users
- Seniors (65+) working on mobility
- Caregivers monitoring progress
- Physical therapists tracking patients

## Key Differentiators
- Camera-based AI form checking
- Personalized voice companion
- No complex app installation
- Home-focused exercise library
